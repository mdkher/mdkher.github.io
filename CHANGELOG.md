# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

- Ongoing content updates to `src/data/portfolio-data.ts`.

## [1.3.0] - 2025-12-25

### Added

- **Case Study Enhancements**: Added support for 'Hero Style' images in Overview and Research sections.
  - New properties: `overviewImages` and `researchImages` in `Project` interface.
- **Documentation**: Created `CASE_STUDY_GUIDE.md` to assist in adding and formatting case study content.

## [1.2.0] - 2025-12-25

### Added

- **Transitions**: Implemented "Geometric Convergence" page transition system using a new directive and service.
- **Navigation**: Introduced a new floating capsule navbar with updated styling and component structure.

### Removed

- **Legacy Components**: Cleaned up obsolete transition components and services to streamline the codebase.

## [1.1.0] - 2025-12-25

### Added

- **Core Visuals**: Implemented custom cursor interaction.
- **Animations**: Integrated GSAP and Lenis for smooth scrolling and complex animations.
- **Styling**: Configured Tailwind CSS for utility-first styling.
- **Deployment**: Added `gh-pages` dependency and deployment scripts for GitHub Pages.

## [1.0.0] - 2025-12-24

### Added

- **Initial Release**: Initialized the Angular portfolio application.
- **Structure**: Set up core components, services, and routing.
